About

鯖江市・福井県について

鯖江市（人口68,284人・面積53.16km²）は眼鏡フレームの国内生産の約96%を担う"ものづくりのまち"。 しかし人口減少と観光客の少なさが課題となっています。

👓 眼鏡のまち・鯖江

国産メガネフレームの約96%を生産。めがねミュージアムなど近未来的なサイバー施設も充実。

🌸 西山公園・きらめきロード

広大な西山公園の桜やつつじ、きらめきロードの雄大な自然。地域住民に愛される公共空間。

⛩️ 歴史的遺産

王山古墳・お寺など歴史的価値の高い史跡が点在。継体天皇・朝倉義景など有名武将ゆかりの地。

🚂 アクセスの課題

JR新宿駅 在来線：1時間26本以上 vs JR福井駅：1時間1〜3本。首都圏からのアクセスが障壁。

Game Concept

アプリ・ゲーム概要

鯖江市・福井県を舞台としたメタバースゲームアプリを作成し、全国のユーザーに鯖江・福井の魅力を伝える。
実際に訪れたいと思わせる体験型プラットフォーム。

⚔️ 心理戦×バトルフィールド

プレイヤーとハンターの2役職が鯖江の街を舞台に激突。タスクをこなして勝利を目指せ。

METAVERSE GAME
🗺️

リアル舞台のメタバース

鯖江市・福井県の実在する場所がバトルフィールドに。プレイするほど土地勘が身につく。

🧠

教育×ゲーム

五教科クイズ＋鯖江クイズをゲームに組み込み。GIGAスクール構想を体現した学習体験。

⚔️

PvP心理戦

対戦色が強く、RPGより幅広いユーザーを巻き込める。コロッセオへ転送されてのPvP決戦。

🏪

企業連携

ロビーで地元企業の商品を購入可能。ゲーム内アイテムで企業商品を宣伝→購買力増加。

🎮

完全無料プレイ

ゲーム課金ゼロ。親御さんも安心の健全ゲーム。教育特化で子供も大人も楽しめる。

🏆

リアルイベント連動

大型連休に鯖江市内でゲーム関連イベントを開催。来場者には限定スキンを配布。

Roadmap

展開ロードマップ

Phase 1
🏗️ チーム結成・開発開始
地元学生でアプリ制作チームを結成。市商工会議所・大学教授をアドバイザーに迎えて開発スタート。クラウドファンディングで資金調達。
Phase 2
🚀 鯖江市でリリース
まず鯖江市を舞台にベータリリース。各種自治体施設に端末を設置、学校・地元企業にビラ配布。友達招待でアバターレベルアップシステム導入。
Phase 3
🌐 福井県全体へ拡大
福井県全域へ舞台を拡大。大型連休にリアルイベントを開催し、来場者に限定スキンを配布。ゲーム内クーポン配布で実店舗来訪を促進。
Phase 4
🏆 北陸・全国へフランチャイズ展開
フランチャイズ方式で他の地方都市へ横展開。地域活性・教育振興・経済活性の三位一体モデルを全国へ。
